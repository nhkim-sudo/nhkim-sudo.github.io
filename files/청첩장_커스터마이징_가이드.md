```javascript
}
```

### 4️⃣ **색상 변경하기**

파일 상단의 `:root` 부분을 찾아 수정하세요:

```css
:root {
    --primary: #FF6B9D;      /* 메인 핑크색 */
    --secondary: #FFC75F;    /* 노란색 */
    --accent: #C9EEFF;       /* 하늘색 */
    --dark: #2D3142;         /* 어두운 회색 */
    --light: #FFF8F0;        /* 밝은 배경색 */
}
```

**색상 코드 찾는 법**: 
- Google에서 "color picker" 검색
- 원하는 색을 선택하고 HEX 코드(#으로 시작)를 복사해서 붙여넣기

예시:
```css
:root {
    --primary: #9B59B6;      /* 보라색으로 변경 */
    --secondary: #3498DB;    /* 파란색으로 변경 */
    --accent: #E8DAEF;       /* 연보라로 변경 */
}
```

## 🔧 웹 프로그래밍 기초 개념 (Python/C와 비교)

### HTML = 구조 (Python의 데이터 구조와 비슷)
```html
<div class="info-item">
    <span>내용</span>
</div>
```
→ Python으로 비유하면: `{"class": "info-item", "content": "내용"}` 같은 구조

### CSS = 디자인 (변수 설정과 비슷)
```css
.info-item {
    color: red;
    font-size: 20px;
}
```
→ Python으로 비유하면: `info_item_style = {"color": "red", "font_size": 20}` 같은 설정

### JavaScript = 동작/기능 (Python 함수와 비슷)
```javascript
function copyAccount(accountNumber) {
    navigator.clipboard.writeText(accountNumber);
}
```
→ Python으로 비유하면: `def copy_account(account_number): ...` 같은 함수

## 📱 모바일에서 테스트하기

### 방법 1: 파일 공유
1. HTML 파일을 이메일로 자신에게 전송
2. 모바일에서 이메일 열기
3. 첨부파일 다운로드 후 브라우저로 열기

### 방법 2: GitHub Pages (무료 호스팅)
1. GitHub 계정 만들기
2. 새 저장소 생성 (예: `my-wedding`)
3. `wedding-invitation.html` 업로드
4. Settings → Pages → 활성화
5. 제공된 URL로 접속!

### 방법 3: 간단한 로컬 서버 (Python 사용!)
```bash
# 같은 폴더에서 실행
python -m http.server 8000

# 그 다음 브라우저에서:
# http://localhost:8000/wedding-invitation.html
```
→ 같은 Wi-Fi에 연결된 모바일에서도 접속 가능!
→ 컴퓨터 IP주소:8000/wedding-invitation.html

## 🆘 문제 해결

### Q: 파일을 열었는데 이상하게 보여요
A: 메모장 대신 VSCode, Sublime Text 같은 코드 에디터 사용 추천

### Q: 수정했는데 변화가 없어요
A: 브라우저 새로고침 (Ctrl + F5 또는 Cmd + Shift + R)

### Q: 모바일에서 글자가 너무 작아요
A: CSS에서 `font-size` 값을 키우세요 (예: `1.1rem` → `1.3rem`)

### Q: 사진이 안 보여요
A: 이미지 파일 경로 확인 (같은 폴더에 있는지, 파일명이 맞는지)
